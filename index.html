Scatterplot that toggles from 2003 to 2015

We visualize San Francisco's crime data from 2003 and 2015 reagarding the areas and the type of crime occured.
Each circle corresponds to a specific area while the radious of the circle is proportional to the number of the total crimes in that area.
Use the toggle button to switch between 2003-2015.
We start with the values from 2003.
Everytime you click to the sentence bellow the data will toggle between 2003-2015.
The axis stay the same in order for us to compare the data from the different years.

GeoData

Here we explore K-means clustering.
The overall idea is that each value of K has an associated view,
where each of the K centroids is shown as a large colored dot, and all GPS points belonging to that centroid are colored accordingly.
With the assigned buttons you can set different K values.
You can also hover above each button to preview the result.
We start with clustering of 2.
The file size used should be small in order to be read faster.
